# sia

<http://dhcmrlchtdj.github.com/sia>

'sia' means 'write' in Fuzhou dialect.

'pieh' means 'pen'.

------

- `_pieh`: application.
- `_source`: markdown source files.
- `post`: published html files.
- `theme`: theme dictionary.
