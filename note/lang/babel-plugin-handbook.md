> The three primary stages of Babel are _parse_, _transform_, _generate_.

+ Parsing
	- Lexical Analysis
	- Syntactic Analysis
+ Transformation
	- Traversal
	- Visitors
+ Code Generation

只看 SICP 4.1 的话，这些东西都会接触到，但可能没有这么清晰的概念。
