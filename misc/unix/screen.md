```
^a ? - help
^a ^d - detach

^a ^w - window list, where am I

^a ^c - create new window
^a ^p - previous window
^a ^n - next window
^a ^a - switch to previous screen (toggle)
^a [0-9] - go to window [0-9]

^a esc - copy mode, which I use for scrollback
```
